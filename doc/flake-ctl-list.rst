FLAKE-CTL-LIST(8)
=================

NAME
----

**flake-ctl list** - List registered flake applications

SYNOPSIS
--------

.. code:: bash

   USAGE:
       flake-ctl list

   OPTIONS:
       -h, --help       Print help information
       -V, --version    Print version information


DESCRIPTION
-----------

List registered flake applications.

FILES
-----

* /usr/share/flakes

EXAMPLE
-------

.. code:: bash

   $ flake-ctl list
